# Text-Classification-and-Sentiment-Analysis-Machine-Learning-Project
Project used to learn how to use Support Vector Machines and Sentiment Analysis to classify data entries.

The dataset used for this project contains speeches from the UN General debates (1970 - 2017).

Database credit goes to https://github.com/blueprints-for-text-analytics-python/blueprints-text/tree/master/data/un-general-debates
__________________________________________________________________________________________________________________

## Text Classification
Using scikit-learn and NLTK I built a text processing and classifier pipeline to predict the country of a speech. In the interest of runtime, I filtered the database to include only speeches that occured from the year 2000 - 2017. I created a pipeline that first transforms the text data into TF-IDF vectors (term frequency â€“ inverse document frequency, a measure of importance of a word to a document in a collection adjusted for the fact that some words appear more frequently in general), then applies SVM (Support Vector Machine) to analyze data for classification while referencing a set of stop words to eliminate words that are widely used and carry very useful information for the purposes of this project.

The rows of the classification report represent the classes (countries) being predicted.

Each row shows the following metrics
- Precision: The ratio of correctly predicted positive observations to the total predicted positives.
- Recall: The ratio of correctly predicted positive observations to all observations in the actual class.
- F1-score: The harmonic mean of precision and recall. It provides a balance between the two metrics.
- Support: The number of actual occurrences of each class in the dataset.

The overall metrics indicate the model performs well with an accuracy of 92%

Many countries, such as Albania (ALB), Argentina (ARG), and Australia (AUS), have high precision, recall, and F1-scores, suggesting that the classifier is effective at identifying these countries.

Some countries, like Germany (DEU) and Ethiopia (ETH), have lower recall values, indicating that while the classifier correctly predicts these countries when it does so, it misses several instances where it should have predicted them.

A few countries, such as Djibouti (DJI), show a precision and recall of 0.00, indicating that the classifier completely failed to identify instances of those countries in the dataset.

The overall trend of high precision and recall scores suggests that the classifier is well-tuned for most countries, but improvements may be needed for specific cases where the classifier struggles.
